## Quick Start

### Prerequisites

- Node.js 20.x or higher
- npm 8.x or higher

### Installation

```bash
# Clone the repository
git clone https://github.com/stusseligmini/CeloraGroup.git
cd CeloraV2

# Install dependencies
npm install

# Configure environment (contact admin for credentials)
cp .env.example .env.local

# Start development server
npm run dev
```

Visit `http://localhost:3000` to see the application.
