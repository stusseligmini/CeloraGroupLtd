# Celora Browser Extension User Guide

## Installation

### Chrome / Edge
1. Visit [Chrome Web Store](https://chrome.google.com/webstore/category/extensions)
2. Search for "Celora Wallet"
3. Click **"Add to Chrome"** / **"Add to Edge"**
4. Confirm by clicking **"Add Extension"**
5. Pin extension to toolbar for easy access

### Firefox
1. Visit [Firefox Add-ons](https://addons.mozilla.org/)
2. Search for "Celora Wallet"
3. Click **"Add to Firefox"**
4. Click **"Add"** to confirm
5. Pin to toolbar (optional)

## Getting Started

### First Launch
1. Click the Celora icon in your toolbar
2. Sign in with your Celora account
3. Extension syncs with your web account
4. You're ready to go!

- Extension locks after 5 minutes of inactivity
- Requires re-authentication
- Configurable timeout

- Separate from main account password

### Biometric Auth
- Use fingerprint/Face ID (if supported)

## Troubleshooting

### Extension Not Loading
1. Reload extension (Extensions page → Reload)
2. Clear browser cache
3. Reinstall extension
4. Check for browser updates

### Sync Issues
1. Check internet connection
2. Sign out and sign back in
3. Force sync (Settings → Sync Now)
4. Contact support if persists

### Authentication Errors
1. Clear browser cookies
2. Sign out and sign in again
3. Reset password if necessary
4. Check for 2FA issues

### Extension Icon Not Showing
1. Check if extension is enabled
2. Pin extension to toolbar manually
3. Restart browser
4. Reinstall if necessary

## FAQs

**Q: Is my data safe in the extension?**
A: Yes! All sensitive data is encrypted locally. Your private keys never leave your device.

**Q: Can I use the extension offline?**
A: View-only features work offline. Transactions require internet connection.

**Q: Does the extension use a lot of resources?**
A: No, the extension is optimized for minimal CPU and memory usage.

**Q: Can I use multiple devices?**
A: Yes! Sign in on multiple devices. Data syncs automatically.

**Q: What permissions does the extension need?**
A: Only permissions necessary for functionality:
- Storage (save settings)
- Active Tab (detect URLs)
- Notifications (alerts)
